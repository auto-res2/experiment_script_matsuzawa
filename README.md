# Information Bottleneck Guided Triplet Graph Transformer for Efficient Molecular Graph Learning
> ⚠️ **NOTE:** This research is an automatic research using AIRAS.
## Abstract
We introduce the Information Bottleneck Guided Triplet Graph Transformer (IBGT), a novel framework that integrates an information bottleneck (IB) mechanism into triplet-based graph transformers to achieve efficient molecular property prediction. In contrast to conventional Triplet Graph Transformers (TGT) that compute exhaustive third-order interactions, IBGT leverages an adaptive gating mechanism to selectively process only those triplets that exhibit high mutual information with the target property. The model accomplishes this by estimating relevance scores for triplet interactions and then discarding those that contribute little to the prediction task. The learnable filtering not only reduces the computational overhead from quadratic or higher complexities toward a more manageable level but also enhances interpretability by isolating chemically meaningful substructures. We evaluate our approach on standard molecular datasets such as QM9, ESOL, MOLPCBA, and others using a three-stage training procedure that includes distance prediction, noisy three-dimensional pretraining, and fine-tuning on predicted distances. Extensive experiments and ablation studies confirm that IBGT achieves competitive or improved predictive performance while significantly reducing runtime and memory consumption compared to the baseline TGT. Interpretability experiments further demonstrate that the filtered triplets correspond to important molecular interactions, offering a promising direction for efficient and transparent graph learning in molecular applications.
